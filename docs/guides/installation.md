# Installation
### Prerequisites
1. Python 3.12 or higher
2. Poetry

### Installation
mkcli can be installed using pip. To install mkcli, run the following command:

```bash
python -m pip install git+https://github.com/CloudFerro/cf-mkcli.git
```

Now you can run it just with:
```bash
mkcli
```
or
```bash
python -m mkcl
```
